# Proiect website personal
Acesta este un website personal și proiectul la materia Tehnici Web 2024-2025. Conține câteva informații personale, nu utilizează niciun framework și constă doar în partea de frontend (HTML, CSS, JavaScript).

## Accesare
Link-ul pentru accesarea site-ului este: [https://a-violeta.github.io/avi_site](https://a-violeta.github.io/avi_site/)

## Functionalități
- Design responsiv
- Meniu dropdown
- Formular
- Logare utilizator
- Animație în CSS
- Schimbare temă site (light mode / dark mode)
